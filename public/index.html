4x2048

TEMPO 00:00 PONTOS 0 MELHOR 0
Una os blocos nos 4 tabuleiros até chegar em 2048 em todos.
Como jogar
Use as setas direcionais para mover os blocos nos tabuleiros. Quando dois blocos com o mesmo valor se tocam, eles mergem em um novo bloco com a soma dos dois blocos anteriores. Você precisa ter um bloco com 2048 em cada tabuleiro para vencer. Caso algum tabuleiro fique sem espaço para uma nova peça ou movimento, você perde.
